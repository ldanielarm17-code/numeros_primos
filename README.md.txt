Validador de Números Primos Neón 🚀

Este es un aplicativo web interactivo y moderno diseñado para verificar si un número entero positivo es primo o no. El proyecto combina una lógica matemática eficiente con una interfaz visual de alto impacto inspirada en la estética Cyberpunk/Neón.

✨ Características

Lógica de Validación: Utiliza un algoritmo optimizado que comprueba divisores hasta la raíz cuadrada del número, manejando eficientemente números grandes.

Interfaz Neón: Diseño responsivo con efectos de desenfoque de fondo (glassmorphism), bordes brillantes y tipografía clara.

Fondo Animado: Números que flotan suavemente en el fondo para mejorar la experiencia visual.

Efectos Visuales Dinámicos:

Celebración: Efecto de confeti verde y cian cuando se identifica un número primo.

Explosión: Efecto de partículas rojas intensas cuando el número no es primo o es un caso especial (0, 1, negativos).

Diseño Responsivo: Adaptable a dispositivos móviles, tablets y computadoras de escritorio.

🛠️ Tecnologías Utilizadas

HTML5: Estructura semántica del aplicativo.

Tailwind CSS: Framework utilizado para el estilizado rápido y profesional, incluyendo efectos de sombras neón y diseño responsivo.

JavaScript (Vanilla): Lógica matemática de los números primos y manipulación del DOM.

Canvas-Confetti: Librería externa para los efectos de partículas y celebraciones.

🚀 Cómo utilizarlo

Abre el archivo validador-primos.html en cualquier navegador web moderno.

Introduce un número entero en el campo de entrada central.

Haz clic en el botón "Verificar Número" o presiona la tecla Enter.

¡Disfruta de la animación resultante y lee la explicación detallada!
